# 카테고리별 작성 가이드

각 카테고리의 특성과 작성 시 유의사항.

## Python

Java 개발자가 Python으로 전환하는 관점에서 작성:
- Type hints, typing 모듈
- OOP 개념 (@dataclass, @abstractmethod, self)
- Python 특화 기능 (f-string, *args/**kwargs, with)

## Java

- 버전별 새 기능 (Java 21, 25 등)
- 모던 Java 패턴

## Spring

- Spring Framework 핵심 개념 (IoC, DI, AOP)
- Spring Boot 설정과 패턴
- 실무 활용 예제

## Node.js

- 비동기 I/O, Event Loop
- libuv, Thread Pool
- Worker Threads, Cluster

## Computer Science

- 동시성, 병렬성
- 동기/비동기, 블로킹/논블로킹
- 알고리즘, 자료구조

## Security

- 보안 개념 (MITM, PII 등)
- 실무 보안 구현
